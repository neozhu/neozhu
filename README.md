## About Me

I'm an independent software developer with **20+ years of software and systems engineering experience**, specializing in modern **.NET and Blazor application development**.

I help teams turn early-stage ideas, complex requirements, and legacy workflows into practical software that is ready for real-world use.

My work focuses on:

* Building complete Blazor applications from concept to production
* Designing clean, scalable, and maintainable application architectures
* Developing reliable backend APIs and integrations
* Modernizing existing .NET applications
* Improving deployment, automation, and production readiness
* Delivering quickly without sacrificing long-term maintainability

I am open to **freelance projects, technical collaborations, and open-source partnerships**.
